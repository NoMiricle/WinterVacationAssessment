### 查看方式
[获取代码](https://github.com/NoMiricle/WinterVacationAssessment)
```
git clone https://github.com/NoMiricle/WinterVacationAssessment
```
* 仅需VScode
* 最好是用chorm浏览器打开网页
* 在html那个文件打开浏览器

除了git那个文件其余文件都下载下来或者复制到VScode文件中
